Nutrition Intuition is a robust full-stack web development project designed to track food nutrition, completed in 10 weeks by a 6-member team. We selected the MERN stack for its user-friendliness and reliability.

As the project manager, I implemented agile methodologies, ensured thorough regular requirement analysis and stakeholder engagement, while avoiding crunch. This project enhanced my front-end react development and NoSQL database management skills, resulting in a versatile platform with web and mobile interfaces.
