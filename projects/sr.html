SLEDDVR

❯

Solo-developed physics based sledding game built with Unity for Meta Quest platform.

SLEDDVR is an immersive VR game featuring innovative movement mechanics and procedural terrain generation.

I independently developed and released SLEDDVR, implementing custom cloud-based leaderboards using Unity Gaming Services, JavaScript, and C#. I created all 3D models, particle effects, and marketing assets resulting in thousands of downloads since release. This project demonstrates my ability to deliver a full-cycle game development project that resonates with players.
