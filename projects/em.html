EasyMeshVR

Open-source VR program for collaborative 3D mesh editing, featuring intuitive hand gestures and real-time file transfers.

1 / 3
Move vertices, edges, and faces by simply reaching out and grabbing them
2 / 3
Hop into a multiplayer lobby to collaborate on your edits using in-game voice chat
3 / 3
Easily transfer 3D model files between VR and your PC using our webserver
❮ ❯

EasyMeshVR, an open-source VR program, was conceived and executed by my team and me within 6 months. This project, developed in Unity, enhances 3D mesh editing using intuitive hand gestures for manipulation. Other notable features include real-time collaborative editing via multiplayer lobbies and efficient 3D model file transfers through our web server.

As project manager, drawing on my experience from Nutrition Intuition, my primary responsibilities were to define project requirements and oversee task progress. This role required my skill as both a creative problem solver and an effective leader.
